TouchEn mVaccine SDK · APP

2019.04 — 현재
메인 담당 (Android/iOS SDK, App) · 라온시큐어

금융·공공 200여 기업의 1,000만+ 사용자 서비스에 탑재되는 모바일 SDK. Android·iOS 동시 운영하며 멀티모듈 아키텍처 설계, Kotlin 99% 전환, 성능 안정화, OEM 호환성 관리를 담당했다.

멀티모듈 아키텍처(core/detection/ui)로 전환하여 호스트 앱과의 의존성 충돌을 줄이고 기능별 독립 빌드·배포 가능한 SDK 구조 설계. Java→Kotlin 99% 마이그레이션은 public API 하위 호환성을 유지하며 완료.
  • 성능 안정화: WorkManager·Coroutines 기반 비동기 처리 도입. JNI 네이티브 메모리 릭까지 추적하여 ANR 0건 달성.
  • IPC 설계: AIDL/Bound Service 기반 프로세스 간 통신(앱 간), URL Scheme 기반 앱-웹 간 통신 기능 개발.
  • OEM 호환성: Android 5~16, 삼성·LG·샤오미·Pixel 단편화 대응. Reflection 기반 제조사·OS별 비공개 API 차이 흡수 레이어 구현.
  • Public API 운영: 업데이트 시 호스트 앱 코드 수정 0건 원칙 기반으로, 200여 기업이 통합하는 SDK의 Public API를 하위 호환성을 유지하며 설계·운영.
  • DX·통합 지원: 연동 가이드·샘플 프로젝트·마이그레이션 문서·릴리즈 노트 작성. 빌드 사이즈·ProGuard/R8 규칙·의존성 충돌 등 호스트 앱 영향 지표 관리.
  • SDK 최적화: 통합→분할 다운로드 전환으로 데이터 사용량 절감. SDK 바이너리에서 패턴·엔진 파일 분리로 사이즈 최적화.
  • 품질 체계: KISA 인증 획득. JUnit 기반 SDK 품질 관리. Jenkins CI/CD 자동화. Firebase Crashlytics/Analytics 원격 모니터링·분석 체계 구축.
  • iOS: 탈옥 탐지, 원격 탐지, 디버깅 탐지 모듈 개발. Objective-C 기반 iOS SDK 유지보수.

  • 영상 업로드 및 유튜브, SNS 등 URL 입력 → 결과 수신 UX 구현.
  • 실시간 딥페이크 탐지 현황 및 랭킹 UI/UX 구현.
  • 50만 다운로드 B2C 앱(라온 모바일 시큐리티)에 탑재 및 실서비스 출시.
  • 이를 바탕으로 KISA 과제로 온디바이스 딥페이크 탐지 SDK 개발 진행 중 — SDK 설계·개발 메인, 모델 경량화(양자화·지식증류) 연구 서브. 인터페이스 추상화 기반 단일 진입점 구조 설계 (모델 백엔드·입출력·다중 탐지기). 제품화 단계 예정.
