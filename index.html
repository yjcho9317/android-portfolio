  • 성능 안정화: WorkManager·Coroutines 기반 비동기 처리 도입. ANR과 메모리 릭을 JNI 레벨(GlobalRef 릭, native allocation)까지 추적·해결하여 내부 QA 기준 ANR 재현 0건 달성.

  • 앱 간·앱-웹 통신: URL Scheme 기반 앱-웹 간 통신 및 AIDL/Bound Service 기반 앱 간 통신 기능 개발.
  • SDK 호환성: Android 8~15(7개 메이저 버전), 삼성·LG·샤오미·Pixel 호환성 검증. Reflection 기반 제조사·OS별 비공개 API 차이 흡수 레이어 구현. 플레이스토어 배포 및 업데이트 운영.

AI 딥페이크 탐지 — 온디바이스 vs 클라우드 아키텍처 전환 판단

2024.04 — 2024.12
앱 개발 리드 / 모델 개발 서브 (6인 팀) · 라온시큐어

오픈소스 딥페이크 탐지 모델을 모바일 보안 서비스에 응용. 앱 개발을 메인으로, 모델 튜닝을 서브로 담당하며 아키텍처 전환을 결정.

자체 데이터셋 구축으로 정확도 30%p 개선(95%). TFLite 양자화(FP32→FP16→INT8) 적용 후 온디바이스 추론 속도와 MediaProjection 해상도 제약사항(720p, 30fps 미달) 확인 → 클라우드 아키텍처로 전환하여 서비스 출시 결정.
  • MediaProjection API로 실시간 프레임 캡처 구현 후, 해상도·프레임레이트 제약을 확인하여 클라우드 아키텍처로 전환.
  • 영상 업로드 및 유튜브, SNS 등 URL 입력 → 결과 수신 UX 구현.
  • 실시간 딥페이크 탐지 현황 및 랭킹 UI/UX 구현
  • 50만 다운로드 B2C 앱(라온 모바일 시큐리티)에 탑재. 앱-서버 간 비동기 통신 구조 및 결과 표시 UI 구현.

  • KoBERT 기반 보이스피싱 탐지 모델 설계 및 학습 파이프라인 구축. 후속 개선 방향으로 정상 대화 데이터 보강 및 Hard Negative Mining 전략 제안.
  • 모델 학습 파이프라인을 구축하고 추론 흐름을 구현. 모델 성능 지표뿐 아니라 체감 품질까지 검증하여 상용화 판단 결정.
